# Portal Cáncer Gástrico — Refactor a múltiples archivos

Esta estructura separa **HTML / CSS / JS** sin cambiar el diseño ni la funcionalidad.

```
portal-cancer-gastrico/
├─ index.html
└─ assets/
   ├─ css/
   │  └─ styles.css
   └─ js/
      └─ app.js
```

## Cómo usar
1. Abre `index.html` en el navegador (no requiere servidor).
2. Asegúrate de mantener las rutas: `assets/css/styles.css` y `assets/js/app.js`.

## Próximo paso
- Te puedo insertar la **SCAR** en `DATA.factores` (dentro de `app.js`) sin tocar el resto.
